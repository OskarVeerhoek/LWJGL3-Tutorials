org.lwjgl.opengl

Class WGLARBPixelFormat

  • java.lang.Object
    • org.lwjgl.opengl.WGLARBPixelFormat

  • public final class WGLARBPixelFormat
extends java.lang.Object
    Native bindings to the WGL_ARB_pixel_format extension.

    This extension adds functions to query pixel format attributes and to choose from the list of supported pixel formats.

    These functions treat pixel formats as opaque types: attributes are specified by name rather than by accessing them directly as fields in a structure. Thus the list of attributes can be easily extended.

    Requires WGL_ARB_extensions_string.

      • wglChoosePixelFormatARB

        public static int wglChoosePixelFormatARB(long hdc,
                          java.nio.ByteBuffer attribIList,
                          java.nio.ByteBuffer attribFList,
                          int maxFormats,
                          java.nio.ByteBuffer formats,
                          java.nio.ByteBuffer numFormats)
        Selects from among all of the available pixel formats (including both accelerated and generic formats and non-displayable formats). This function accepts attributes for the main planes. A list of pixel formats that match the specified attributes is returned with the "best" pixel formats at the start of the list (order is device dependent).
        Parameters:
        hdc - the device context on which the pixel format is supported
        attribIList - a list of attribute {type, value} pairs containing integer attribute values. One of:
        NUMBER_PIXEL_FORMATS_ARB, DRAW_TO_WINDOW_ARB, DRAW_TO_BITMAP_ARB, ACCELERATION_ARB, NEED_PALETTE_ARB, NEED_SYSTEM_PALETTE_ARB, SWAP_LAYER_BUFFERS_ARB, SWAP_METHOD_ARB, NUMBER_OVERLAYS_ARB, NUMBER_UNDERLAYS_ARB, TRANSPARENT_ARB, TRANSPARENT_RED_VALUE_ARB, TRANSPARENT_GREEN_VALUE_ARB, TRANSPARENT_BLUE_VALUE_ARB, TRANSPARENT_ALPHA_VALUE_ARB, TRANSPARENT_INDEX_VALUE_ARB, SHARE_DEPTH_ARB, SHARE_STENCIL_ARB, SHARE_ACCUM_ARB, SUPPORT_GDI_ARB, SUPPORT_OPENGL_ARB, DOUBLE_BUFFER_ARB, STEREO_ARB, PIXEL_TYPE_ARB, COLOR_BITS_ARB, RED_BITS_ARB, RED_SHIFT_ARB, GREEN_BITS_ARB, GREEN_SHIFT_ARB, BLUE_BITS_ARB, BLUE_SHIFT_ARB, ALPHA_BITS_ARB, ALPHA_SHIFT_ARB, ACCUM_BITS_ARB, ACCUM_RED_BITS_ARB, ACCUM_GREEN_BITS_ARB, ACCUM_BLUE_BITS_ARB, ACCUM_ALPHA_BITS_ARB, DEPTH_BITS_ARB, STENCIL_BITS_ARB, AUX_BUFFERS_ARB
        attribFList - a list of attribute {type, value} pairs containing floating point attribute values
        maxFormats - the number of attributes being queried
        formats - an array of returned indices of the matching pixel formats. The best pixel formats (i.e. closest match and best format for the hardware) are at the head of the list.
        numFormats - returns the number of matching formats
