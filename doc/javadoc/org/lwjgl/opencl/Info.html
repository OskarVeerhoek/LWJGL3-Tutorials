org.lwjgl.opencl

Class Info

  • java.lang.Object
    • org.lwjgl.opencl.Info

  • public final class Info
extends java.lang.Object
    This class provides methods that can be used to query information about OpenCL objects. These methods are similar to the corresponding clGet&lt;Type&gt;Info function for each object type, except that only a single value is returned. Which one of these methods should be used depends on the type of the information being queried.
