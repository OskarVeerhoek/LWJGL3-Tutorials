        public static int clock_getres(int clk_id,
               java.nio.ByteBuffer res)
        Finds the resolution (precision) of the specified clock clk_id, and, if res is non-NULL, stores it in the timespec struct pointed to by res. The resolution of clocks depends on the implementation and cannot be configured by a particular process.
        Parameters:
        clk_id - the clock
        res - a pointer to a timespec structure
